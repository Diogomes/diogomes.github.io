Origem & audiência

Ative os gráficos de origem e audiência

País, referrers, navegador, sistema e dispositivo só vêm pela API autenticada do GoatCounter. Para mostrá-los aqui, gere um token de leitura:

  1. No GoatCounter: Settings → Password, API, etc. → API tokens → Create (marque só “Read statistics”).
  2. Cole o token em assets/js/dashboard-config.js (campo token).
  3. Commit/push — os gráficos abaixo passam a carregar.

⚠️ Como esta página é pública (mesmo com noindex), o token fica visível no código-fonte. Use um token somente de leitura. Os dados expostos seriam apenas suas estatísticas de tráfego.
